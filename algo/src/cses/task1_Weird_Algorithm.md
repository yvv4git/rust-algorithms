# Weird Algorithm

**Ограничение времени:** 1 секунда  
**Ограничение памяти:** 256 Мб  
**Ввод:** стандартный ввод или input.txt  
**Вывод:** стандартный вывод или output.txt  

Рассмотрим алгоритм, который принимает на вход положительное целое число n. Если n четное, алгоритм делит его на два, а если n нечетное, алгоритм умножает его на три и прибавляет единицу. Алгоритм повторяет это, пока n не станет равным единице. Например, последовательность для n=3 выглядит следующим образом: 3 → 10 → 5 → 16 → 8 → 4 → 2 → 1  

Ваша задача — симулировать выполнение алгоритма для заданного значения n.  

## Формат ввода  
Единственная строка ввода содержит целое число n.  

## Формат вывода  
Выведите строку, содержащую все значения n во время выполнения алгоритма.  

## Ограничения  
1 ≤ n ≤ 10^6  

## Примеры  

### Пример 1  
**Ввод**  
3  

**Вывод**  
3 10 5 16 8 4 2 1  

### Пример 2  
**Ввод**  
1  

**Вывод**  
1  

## Solution  
```rust
use std::io::{self, BufRead};

fn main() {
    let stdin = io::stdin();
    let mut lines = stdin.lines();

    let n: u64 = lines.next().unwrap().unwrap().trim().parse().unwrap();

    let mut current = n;
    let mut result = vec![current];

    while current != 1 {
        if current % 2 == 0 {
            current /= 2;
        } else {
            current = current * 3 + 1;
        }
        result.push(current);
    }

    for (i, &num) in result.iter().enumerate() {
        if i > 0 {
            print!(" ");
        }
        print!("{}", num);
    }
    println!();
}
