# G. Марсианское ДНК

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 256 Мб
**Ввод:** стандартный ввод или input.txt
**Вывод:** стандартный вывод или output.txt

Вам нужно решить задачу секвенирования ДНК необычного организма. Его ДНК является двумерным и представляет собой таблицу n×m, в каждой ячейке которой находится один из символов «A», «G», «C», «T». Строки таблицы пронумерованы от 1 до n сверху вниз, а столбцы — от 1 до m слева направо. Обозначим символ, расположенный в ячейке на пересечении i-й строки и j-го столбца как a_{i,j}.

Будем называть горизонтальным фрагментом ДНК любую непустую подстроку любой из строк соответствующей таблицы. Более формально, горизонтальным фрагментом называется строка a_{i,l} a_{i,l+1} …a_{i,r} для любых 1 ≤ i ≤ n и 1 ≤ l ≤ r ≤ m.

Аналогично, вертикальным фрагментом ДНК называется любая непустая подстрока любого из столбцов соответствующей таблицы. Более формально, вертикальным фрагментом называется строка a_{l,j} a_{l+1,j} …a_{r,j} для любых 1 ≤ l ≤ r ≤ n и 1 ≤ j ≤ m.

Будем говорить, что строка является фрагментом ДНК, если она является его горизонтальным фрагментом или его вертикальным фрагментом.

В результате эксперимента, проведенного над ДНК данного организма, вы получили большое количество образцов — строк, состоящих из символов «A», «G», «C», «T». Некоторые из этих образцов являются фрагментами данного ДНК, а некоторые — нет.

Ваша задача заключается в том, чтобы по данному набору образцов получить возможную структуру ДНК, из которой данные образцы были получены. При этом вы хотите максимизировать количество образцов, которые являются фрагментами этой ДНК.

Напишите программу, которая по данному набору образцов восстановит любую возможную структуру ДНК. Ваша программа получит количество баллов, пропорциональное количеству образцов, которые являются фрагментами восстановленной вами ДНК.

## Формат ввода
Первая строка входных данных содержит три целых числа n, m и t — размеры искомой ДНК и количество данных вам образцов (1 ≤ n, m ≤ 50; 1 ≤ t ≤ 10000).

Следующие t строк содержат описания данных вам образцов. Каждое описание является строкой длины не более 50, состоящей из символов «A», «G», «C», «T».

## Формат вывода
Выведите n строк по m символов в каждой — восстановленную структуру ДНК. Все строки должны состоять только из латинских букв «A», «G», «C», «T».

## Система оценивания
Ваш счет за каждый тест в данной задаче будет равняться \frac{k}{t}, где k — количество данных образцов, которые являются фрагментами восстановленной вами ДНК. 
Таким образом, счет за каждый тест является вещественным числом от 0 до 1.

Общий счет за задачу будет равен сумме счетов за каждый тест. Всего в данной задаче 100 тестов.

## Пример

**Ввод**
3 3 5
TT
ATT
ACT
GG
AGC

**Вывод**
AGC
CGA
TTT

## Примечания
Приведенная в тесте из условия структура ДНК содержит фрагменты «TT», «ACT», «GG» и «AGC», но не содержит фрагмент «ATT». Таким образом, данный ответ получит \frac{4}{5} балла.
