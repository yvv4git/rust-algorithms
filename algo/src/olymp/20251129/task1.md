# A. Олимпиада и конференция

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 256 Мб
**Ввод:** стандартный ввод или input.txt
**Вывод:** стандартный вывод или output.txt

Событий много, а дней мало! Вот и в этот раз командная олимпиада совпала с конференцией. В каждой команде на олимпиаде должно быть три человека: два программиста и один математик. На конференцию должно пойти не меньше C человек (неважно, кем они будут). Всего в компании работают P программистов и M математиков. Определите максимальное количество команд, которое можно сформировать с учётом того, что часть сотрудников пойдёт на конференцию.

## Формат ввода
Вводится три числа: P, M, C (0 ≤ P, M, C ≤ 10^18) — количество программистов, математиков и людей, которые должны пойти на конференцию. Гарантируется, что P + M ≥ C.

## Формат вывода
Выведите одно число — максимально возможное количество сформированных команд.

## Система оценивания
Решения, верно работающие в случаях, когда все числа не превосходят 100, будут получать не менее 50 баллов.

## Примеры

### Пример 1
**Ввод**
6 3 2

**Вывод**
2

### Пример 2
**Ввод**
2 1 1

**Вывод**
0

### Пример 3
**Ввод**
6 10 3

**Вывод**
3




## Solution
use std::io;

fn main() {
    let mut input = String::new();
    io::stdin().read_line(&mut input).unwrap();
    let nums: Vec<u128> = input
        .split_whitespace()
        .map(|x| x.parse::<u128>().unwrap())
        .collect();

    let (p, m, c) = (nums[0], nums[1], nums[2]);

    let t1 = p / 2;                 // ограничение по программистам
    let t2 = m;                     // ограничение по математикам
    let t3 = (p + m - c) / 3;       // чтобы осталось >= C человек

    let result = t1.min(t2).min(t3);

    println!("{}", result);
}
