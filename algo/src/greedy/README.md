# D. Автосалон

Не решена

## Ограничение времени
1 секунда

## Ограничение памяти
256Mb

## Ввод
стандартный ввод или input.txt

## Вывод
стандартный вывод или output.txt

На складе автосалона хранится n автомобилей. Цвет каждого автомобиля задается числом.

В выставочном зале необходимо разместить k автомобилей. Чтобы привлечь покупателей было принято решение выставить такие k автомобилей, чтобы количество различных цветов автомобилей было максимальным.

## Формат ввода
В первой строке вводится два целых числа n,k (1≤k≤n≤100000) — количество автомобилей на складе и количество автомобилей, которые должны быть в выставочном зале.

В следующей строке дано n чисел ai (1≤ai≤10^9) — цвет автомобиля номер i.

## Формат вывода
Выведите ровно k целых чисел через пробел — цвета автомобилей, которые нужно выставить в зале.

Если правильных ответов несколько — выведите любой из них.

## Пример 1
### Ввод
```
5 3
1 2 1 2 1
```
### Вывод
```
1 2 1
```

## Пример 2
### Ввод
```
10 4
2 1 8 8 8 8 8 8 8 8
```
### Вывод
```
1 2 8 8
